Free yourself with free software

Discover freedom

What is software?

You are using software right now to view this website. You probably already know what software is, and may refer to it as an app or a program. You use software everyday to send emails, write documents, browse the internet, play computer games, and more.


What do you mean, free?

It may sound confusing, but free software does not mean free of cost. On the contrary, free software means free as in freedom. When you use free software there are no restrictions involved. Free software ensures a number of freedoms to you when you use it. You are free to use it however you want, share it with whoever you want, and modify it to do whatever you want.




Freedoms Ensured by Free Software


Use Software However You Want

There are no restrictions with how you can use free software. You may use the software on any computer you want. There is no time limit on how long you are able to use the software. You get to control what you do with the software, not the corporation that made it.


Share Software With Whoever You Want

With free software, you are free to share a copy with a friend, family member, or stranger. You may share the software with as few or as many people as you like. If you share free software with someone, they are entitled to all the same freedoms that you were.


Modify Software To Do What You Want

If you are not a programmer, it is unlikely that you will be able to modify free software yourself. Don't let that stop you! You can make requests to programmers to improve the software you use. It is easier than you might think. It is very common for free software to have online communities where you can make requests.

Why you should care

Free software has many benefits

You Get Better Software

Software that isn't free software is worked on only by the corporation that owns it. The number of people who are able to improve it is limited. This is not the case for free software. There are no restrictions on who can work on free software, so programmers all over the world are able to make improvements. Since the programmers working on free software are often users as well, they are dedicated to make the software better. With so many eyes on the software, it will be less buggy.


You Get Trustworthy Software

Free software is inspected by programmers all over the world. These programmers aren't just improving the software, they are auditing it as well. Not all programmers are honest about the software they make. Some software will claim to do one thing, but do something else you don't expect. It might play music like it's supposed to, but it's secretly spying on you and stealing your credit card information.
This kind of thing doesn't happen with free software. If the software does anything it's not supposed to, it won't be long before someone finds out. The transparency of free software keeps programmers honest, which leads to more trustworthy software.


The dangers of non-free software

Software that works against you

What is Non-Free Software?

Non-free software is restrictive. You are not gaurenteed the freedom to use non-free software any way you want, share it with anyone you want, or modify it to do what you want. The corporation that makes the software tells you what you can and can't do with it.



The Dangers


Invasion of Privacy

Non-free software might be spying on you without you knowing it. Since it is not possible to inspect non-free software, it is very difficult to know what it is doing behind the scenes. It may be snooping through your personal data and collecting information to sell to advertisers.


Security Issues

The creators of non-free software might leave backdoors in their software. A backdoor is a hole left in software security so that it can later be used to gain access to a computer without the users knowledge. It's bad enough that the programmer who created that backdoor could gain access to your computer, but if a clever hacker discovered the backdoor, they would be able to gain access too.
Since it is not possible to inspect non-free software, it can be very difficult to know whether there are backdoors hidden in it. Even if they are discovered, they can not be removed, because non-free software cannot be modified.




Examples


Spotify

It was discovered that the music app, Spotify, was spying on it's users without their knowledge. Spotify was snooping through peoples pictures, media files, and contacts. The spying was supposedly being done to provide better advertisements to users.
Spotify is non-free software so there is no way for people to inspect the software to keep its programmers honest.


Microsoft Windows

In 2013, the documents detailing the United States NSA PRISM surveillance program were leaked. It was discovered that the NSA made a deal with Microsoft and other companies to place backdoors in their software. A backdoor is a hole left in software security so that it can later be used to gain access to a computer without the users knowledge. Microsoft put these backdoors into their product, Windows. Not only can Microsoft or the United States government gain full access to a computer running Windows, it's possible that a clever hacker could discover a way to do it too.
Microsoft Windows is non-free software, so Microsft was able to get away with placing backdoors in their software without anyone noticing. Since there is no way for people outside of Microsoft to inspect or modify Windows, there is no way to know whether the backdoors still exist, and there is no way to remove them.


make the switch

Start using free software today

Xubuntu

Xubuntu is a great free software alternative to Windows or MacOS. Xubuntu is a popular linux distribution. It is easy to install and easy to use.


Mozilla Firefox

Firefox is a popular web browser. It is a great free software alternative to Google Chrome, Internet Explorer, Microsoft Edge, and Safari. Firefox is faster and more lightweight than any other web browser on the net.


LibreOffice

LibreOffice is a great free software alternative to Microsoft Office. It is easy to use, and it is fully compatible with all Microsft Office documents.


VLC

VLC is a popular media player. It is a great free software alternative to Windows Media Player and QuickTime. VLC is able to play any music or video file, and can be used to help organize your media library.
